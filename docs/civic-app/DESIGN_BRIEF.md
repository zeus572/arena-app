# Design Brief — Youth-First Civic Sensemaking Platform

## Working Name
Use placeholder names during exploration. Strong candidates:
- Civic Spark
- Public Lab
- The Why Brief
- How Power Works
- Common Reason
- Civic Lens
- Future Citizen

## Core Wedge
**Current events that teach young people how power works.**

## Mission
Help teens and young adults understand current events, civic institutions, public decisions, and their own developing beliefs without dumbing public life down.

## Product Category
A youth-first civic learning and sensemaking platform.

Not just:
- kids’ news
- a civics textbook
- a classroom worksheet site
- a partisan commentary product
- a media-bias comparison site
- a static pro/con debate library

Instead:
- live current events
- civic vocabulary
- institution maps
- values-in-conflict prompts
- issue timelines
- quizzes
- social-native explainers
- teacher/parent trust layer

## Primary Audience
### Middle school / early teens
Needs:
- simple, visual explainers
- civic vocabulary
- short quizzes
- “who has power here?” maps
- safe, non-combative prompts

### High school students
Needs:
- current events with background
- argument comparison
- media literacy
- issue timelines
- classroom-friendly discussion prompts

### College students and young adults
Needs:
- deeper tradeoffs
- institutional reasoning
- belief/value reflection
- issue following
- primary-source links

## Secondary Audience
- teachers
- parents
- homeschool families
- debate clubs
- student government groups
- civic educators

## User Promise
Users should leave feeling:
- “I understand what happened.”
- “I know which part of government is involved.”
- “I learned a civic concept.”
- “I understand why people disagree.”
- “I can think more clearly about my own view.”

## Brand Attributes
The product should feel:
- current
- trustworthy
- youthful but not childish
- visual
- smart
- respectful
- clear
- intellectually honest
- social-native
- classroom-safe

Avoid:
- preachy
- partisan
- condescending
- sterile
- adult-news-heavy
- generic AI-summary feel
- textbook-like dryness
- outrage-first design

## Key Product Modules
### 1. Latest Civic Briefings
Short explainers tied to current events.

Each card should show:
- headline
- institution involved
- 30-second summary
- why it matters
- key concept
- “Think deeper” question

### 2. Civic Briefing Detail Page
A structured current-event page.

Sections:
- 30-second version
- 3-minute version
- who acted?
- what changed?
- why it matters
- words you need to know
- what people disagree about
- strongest arguments
- values in conflict
- think deeper
- where to go next

### 3. Learn the System / Concept Library
Evergreen civic vocabulary and institutional concepts.

Examples:
- filibuster
- certiorari
- precedent
- judicial review
- federalism
- continuing resolution
- committee hearing
- agency rulemaking
- executive order
- injunction

### 4. Think Deeper / Values-in-Conflict
Interactive reflection.

Prompts:
- What is your first reaction?
- Which values matter here?
- What would change your mind?
- What might each side miss?
- Can both things be true?

### 5. Quizzes
Short active-learning interactions.

Examples:
- Which branch acted here?
- Is this a fact, interpretation, prediction, or value judgment?
- Which civic concept explains this story?
- Can you explain this headline in 3 sentences?

### 6. Timelines
Visual process pages for:
- a bill
- a Supreme Court case
- an agency rule
- a constitutional doctrine
- an election process

### 7. Teacher / Parent Layer
Trust-building and classroom use.

Features:
- classroom discussion prompt
- printable guide
- parent conversation starter
- source transparency
- age-appropriate notes

## Preferred Visual Directions to Explore
### Direction A: Bright Social-Native
For middle/high school.
- colorful cards
- carousel feel
- quizzes
- streaks
- badges
- energetic illustrations

### Direction B: Civic Intelligence Dashboard
For older teens and college students.
- clean, modern, information-rich
- briefing dashboard
- timelines
- institution labels
- sharp typography

### Direction C: Classroom-Trusted
For teachers and parents.
- calm
- credible
- structured
- clear source transparency
- less gamified

### Direction D: “How Power Works”
Process-driven and diagrammatic.
- institution maps
- power flows
- decision trees
- timelines
- branch-of-government visual language

### Direction E: Editorial Youth Magazine
More like a modern digital publication.
- strong hero stories
- polished cards
- readable explainers
- social snippets
- less app-like

## Design Requirements
- Mobile-first
- Responsive
- Clear hierarchy
- Short cards that expand into depth
- Strong use of tags: Congress, Supreme Court, Executive, Rights, Elections, Process
- Prominent “Who has power here?” element
- Prominent “Think deeper” element
- Visible parent/teacher trust signal
- Mocked data only
- No backend required for visual exploration

## Suggested Taglines
- Current events that teach you how power works.
- Civics for the world you actually live in.
- Know what happened. Understand why. Think for yourself.
- Understand public life without dumbing it down.
- Learn the system behind the headlines.

## Design Success Criteria
A good mockup should make the product feel:
- more engaging than a civics textbook
- more trustworthy than TikTok commentary
- more civic-specific than general kids’ news
- more reflective than partisan content
- more modern than classroom worksheets
